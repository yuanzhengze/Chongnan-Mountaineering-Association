这个设计稿包含了 5 个核心界面，按照玩家的解谜顺序排列。

📂 项目名称：Project 404 Camp
当前状态： 概念设计稿 v1.0 核心机制： 玩家通过浏览看似正常的旧网页，利用开发者工具、鼠标交互和 URL 修改，揭开隐藏在代码背后的恐怖真相。

🟢 流程概览
UI-01 首页 -> (点击成员介绍) -> UI-02 队员档案 -> (发现隐藏入口) -> UI-03 系统拦截(403) -> (破解密码) -> UI-04 内部BBS -> (触发崩坏) -> UI-05 404深渊

📄 UI-01: 首页 (The Facade)
URL: index.html 视觉风格： Web 1.0 复古风格。窄屏布局，宋体字，背景是粗糙的纸张纹理。顶部有一个计数器显示“访问人数”。

1. 界面内容
Header: 校徽 + “崇南大学登山社 (Chongnan Mountaineering Club)”

置顶公告 (红色加粗):

【重要通知】关于暂停2014年暑期登山活动的说明 鉴于近期哀牢山余脉气候异常及不可抗力因素，校团委决定即日起暂停登山社一切野外考察活动。原定于7月12日的“未名峰”科考队请立即归档数据并停止更新。 发布时间：2014-07-20 | 管理员：Admin

侧边栏: “友情链接”、“社团相册”、“联系我们（邮件图标）”。

正文区域:

欢迎来到崇南大学登山社！我们要去的地方，没有路；我们要见的山，在云端。 (下方是一张大合照，5个年轻人背着登山包站在校门口)

2. 隐藏线索 (ARG 机制)
图片异常: 页面停留 30 秒后，大合照会发生微小的 CSS glitch (故障抖动)。如果玩家把图片下载下来调亮，会发现合照背景的教学楼窗户里，有一张巨大的人脸。

Footer (页脚):

Copyright © 2014 CMU-Club. All Rights Reserved. <!-- 系统提示：服务器连接不稳定，检测到来自 IP [玩家IP] 的异常访问。救救我。 --> (注：“救救我”三个字颜色极浅，肉眼几乎看不见，需要全选文字才能高亮显示)

URL: members.html 视觉风格： 类似档案卡的布局。每个人一张证件照和一段简介。

1. 界面内容
标题: 2014暑期科考队成员名单

卡片 1 (队长):

姓名： 林XX 职责： 领队、导航 简介： 三年级生，拥有丰富的雪山攀登经验。

姓名： 苏XX 职责： 摄影、医疗 简介： “记忆可能会骗人，但镜头不会。”

卡片 3, 4: (普通队员，略)

卡片 5 (异常点):

姓名： [数据损坏] 职责： [数据损坏] 简介：

2. 隐藏线索 (ARG 机制)
交互: 当鼠标悬停（Hover）在队长林XX的照片上时，图片会瞬间替换成一张雪盲症视角的模糊照片（全白，只有黑色的手印）。

控制台 (Console): 玩家按下 F12 打开开发者工具，Console 里会不断打印报错信息：

Error: Member ID_05 does not exist. Error: Do not trust the count. Log: 备份入口已转移至 /admin_login

📄 UI-03: 403 Forbidden (The Gate)
URL: admin_login.html (由 UI-02 的线索推导得出) 视觉风格： 标准的 Apache 服务器错误页面，白底黑字，非常简陋。

1. 界面内容
大标题: 403 Forbidden

正文:

You don't have permission to access /private_logs/ on this server.


Apache/2.4.7 (Ubuntu) Server at www.chongnan-climb.edu Port 80

输入框 (伪装): 页面中间突兀地出现一个简陋的 <input> 框，旁边写着“身份验证”。

2. 隐藏线索 (ARG 机制)
解谜: 密码不是数字，而是**“问题”**。

交互: 如果输入错误的密码，弹窗提示不是“密码错误”，而是：“别在这时候喊名字！”

正确答案: 输入在 UI-01 合照的文件名中找到的一串日期（例如 20140715）。输入正确后，页面跳转。

📄 UI-04: 内部 BBS / 登山日志 (The Truth)
URL: log_view.php?id=last_entry 视觉风格： 这是一个**“正在崩坏”**的页面。CSS 样式丢失了一半，文字排版混乱。背景是黑色的，文字是绿色的终端风格。

1. 界面内容
日志列表 (倒序排列):

Entry #07 [2014-07-19]:

我们决定下撤了。雾太大，指南针失灵了。小张说他看见了营地，但那里只有悬崖。

Entry #08 [2014-07-20]:

不要相信地图。地图是活的。

Entry #09 [2025-01-01] (未来/现在的时间):

(此条目正在实时输入...) 你能看到这行字吗？如果你能看到，说明窗口期打开了。 这里的雪是热的。 我们没有死，我们只是... 走不出这个网页。

2. 隐藏线索 (ARG 机制)
动态文本: 最后一篇日志的内容会根据玩家的系统时间自动变化。

音频触发: 页面加载完毕后，会自动播放一段白噪音（Wind.mp3）。白噪音里混杂着摩斯电码，翻译过来是：“不要回头看（DON'T LOOK BACK）”。

📄 UI-05: 404 Not Found / 深渊 (The Horror)
URL: (玩家试图点击日志里的任何链接，或者试图关闭页面时触发) 视觉风格： 纯黑页面。屏幕中央只有一行红色的字。

1. 界面内容
文字内容:

404 Not Found



The requested URL /escape was not found on this reality.

规则类怪谈 (随着页面滚动逐渐显现):

规则 1：如果你看见屏幕反光里有两个人，请保持不动。


规则 2：不要试图关闭浏览器，它在观察你的鼠标轨迹。


规则 3：把身后的门锁好。

2. 隐藏线索 (ARG 机制 - 结局)
打破第四面墙: 网页请求获取玩家的摄像头权限（仅做弹窗吓唬，不真的获取）。

最终链接: 页面最底部有一个极其微小的链接，点击后跳转回 UI-01 首页，但这次首页的所有人脸都被涂黑了，置顶公告变成了：“欢迎加入崇南大学登山社。”（循环开始了）
