# 崇南大学登山社 ARG 项目结构说明书

本文档旨在为新加入的开发者或玩家提供项目的完整架构概览、页面功能详解及核心线索分布指南。本项目构建了一个双层互动的 ARG（侵入式虚拟现实游戏）体验，分为“表世界”（现代化官方网站）和“里世界”（被遗忘的旧社团主页）。

## 1. 项目概览

*   **项目名称**: 崇南大学登山社失踪事件 ARG
*   **核心叙事**: 围绕 2014 年崇南大学登山社在未名峰发生的神秘失踪事故展开。
*   **体验流程**: 玩家从现代化的“表世界”官网进入，发现异常线索，通过隐藏入口进入“里世界”旧网站，解开谜题，挖掘真相。

## 2. 文件结构与架构

```text
/
├── index.html      # [表世界] 崇南大学官方主页 (入口)

├── weather.html         # [表世界] 未名峰气象站数据页
├── history.html         # [表/里] 社团历史页 (连接点)
├── gallery.html         # [表/里] 活动相册页 (连接点)
├── archive.html           # [里世界] 2014年旧社团主页 (ARG核心)
├── members.html         # [里世界] 队员档案页
├── admin_login.html     # [里世界] 管理员登录/403页面

└── assets/images/       # 图片资源 (含关键线索图)
```

## 3. 页面详细说明

### A. 表世界 (Surface World) - 伪装层

这部分模拟了一个正常的、现代化的大学官方网站，用于建立真实感并埋藏通往里世界的入口。

#### 1. 崇南大学官网 (`index.html`)
*   **风格**: 现代、响应式、明亮。
*   **内容**: 校园新闻、通知公告、页脚信息。
*   **核心线索**:
    *   **隐藏入口**: 页脚处的 "Archived Site: CMC_Legacy_2014" 链接，指向 `archive.html`。
    *   **新闻归档**: 校园新闻中有一条关于 "2014年档案数字化通知"，暗示旧数据的存在。

#### 2. 社团历史 (`history.html`)
*   **风格**: 复古时间轴设计。
*   **内容**: 记录社团从成立到暂停活动的历程。
*   **核心线索**:
    *   **2014年8月节点**: 文本被 "涂黑" (Redacted)，鼠标悬停或查看源码可发现 "搜救停止，无生还者" 的残酷真相。
    *   **历任社长**: 2014 年社长信息缺失。

#### 3. 活动相册 (`gallery.html`)
*   **风格**: 瀑布流图片展示。
*   **内容**: 社团日常活动照片。
*   **核心线索**:
    *   **异常照片**: 混入了 `team_photo_2014.png` (合照) 和 `map_sketch.png` (手绘地图)。
    *   **Glitch 效果**: 某些照片在特定交互下会闪烁，显示出 `broken_face` 等恐怖画面。

#### 4. 中国登山协会合作页 (`cma.html`)
*   **风格**: 严肃、官方文档风格。
*   **内容**: 合作协议、安全教育。
*   **核心线索**:
    *   **事故通报**: 包含一份关于 2014 年未名峰事故的官方定性报告，暗示事故原因非自然因素。

#### 5. 未名峰气象站 (`weather.html`)
*   **风格**: 数据仪表盘，科技感但带故障风。
*   **内容**: 实时与历史气象数据。
*   **核心线索**:
    *   **数据异常**: 2014年7月15日（事故当天）的数据显示辐射值 (`Radiation`) 和温度出现极不自然的峰值 (`CRITICAL SPIKE`)。

---

### B. 里世界 (Inner World) - 核心层

这部分是 2014 年停止更新的旧网站，充满了故障艺术 (Glitch Art)、恐怖氛围和解谜要素。

#### 1. 旧社团主页 (`archive.html`)
*   **风格**: Web 1.0 复古风，黑色背景，绿色终端字体，伴有画面撕裂效果。
*   **功能**: 导航至其他里世界页面。
*   **机制**: 页面会随机出现 "DON'T LOOK" 或 "HELP ME" 的控制台日志和弹幕。

#### 2. 队员档案 (`members.html`)
*   **内容**: 2014 年登山队成员列表（队长、医疗、通信等）。
*   **交互**:
    *   鼠标悬停在成员头像上时，图片会瞬间替换为 `broken_face_male.png` 或 `broken_face_female.png` (扭曲的人脸)。
    *   状态栏显示 "MISSING" (失踪) 或 "DECEASED" (确认死亡)。

#### 3. 管理员登录 (`admin_login.html`)
*   **伪装**: 显示为 403 Forbidden 页面。
*   **谜题**: 实际上是一个伪装的登录口。
*   **解法**: 需要输入特定的访问代码（线索散布在其他页面，如控制台输出）。

#### 4. 内部日志 (`log_view.html`)
*   **风格**: 纯文本终端界面，打字机效果。
*   **内容**: 登山队在失联前的最后几篇日记。
*   **叙事**: 揭示了队员们在山上的心理变化和遭遇的超自然现象。

#### 5. 404深渊 (`escape.html`)
*   **风格**: 极简、压抑、无限循环。
*   **机制**: 规则类怪谈风格。页面包含一些奇怪的规则（"不要回头"、"保持安静"）。
*   **陷阱**: 试图点击 "返回" 可能会导致页面刷新但无法离开，模拟被困境地。

## 4. 资源与素材说明

*   `assets/images/School_Emblem.png`: 崇南大学校徽。
*   `assets/images/team_photo_2014.png`: 正常的登山队合影。
*   `assets/images/team_photo_2014_withface.png`: 处理过的合影，暗示灵异现象。
*   `assets/images/broken_face_*.png`: 用于惊吓效果的损坏人脸素材。
*   `assets/images/map_sketch.png`: 带有标记的手绘地图，指向事故地点。

## 5. 快速开始

1.  启动本地服务器: `python3 -m http.server 8080`
2.  访问入口: `http://localhost:8080/index.html`
3.  寻找线索，开启解谜之旅。
